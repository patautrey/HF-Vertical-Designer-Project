Quick Start Guide

A fast operational cheat sheet for the HF Tools Suite.

🚀 Getting Started

  1. Open the HF Tools Suite.
  2. Select a tool from the left sidebar.
  3. Enter values into every field.

All fields must have an entry, even if it is 0. Blank fields stop the calculation engine.

  1. Click Calculate.
  2. Review results in the right panel.
  3. Click Download Results to save a text file.

🛠 Tool Summary

  • Vertical Designer: Resonant length, NVIS, DX tendency
  • Feedline & Matching: Loss, SWR transformation
  • DX Patterns: Takeoff angle, DX potential
  • NVIS Field: NVIS angle, coverage radius
  • Doublet Designer: Impedance & tuner recommendations
  • Random Wire: Avoid lengths & match type

📄 Exporting Results

Every tool exports a text file named: 

KG5IEF - <Tool Name>.txt

❗ Common Mistakes

  • Leaving a field blank → enter 0
  • Not pressing Calculate before exporting
  • Entering MHz values incorrectly (e.g., “7.” instead of “7”)
