Quick Start Guide

A fast operational cheat sheet for the HF Tools Suite.

🚀 Getting Started

  1. Open the HF Tools Suite in your browser.
  2. Select a tool from the left sidebar.
  3. Enter values into every field.

All fields must have an entry, even if it is 0.
Blank fields stop the calculation engine.

  1. Click Calculate.
  2. Review results in the right panel.
  3. Click Download Results to save a text file.

🛠 Tool Summary

  • Vertical Designer: Resonant length, NVIS angle, DX tendency
  • Feedline & Matching: Feedline loss, delivered power, transformed SWR
  • DX Pattern Explorer: Takeoff angle, lobe classification, DX potential
  • NVIS Field Estimator: NVIS angle, coverage radius, suitability
  • Doublet Designer: Impedance tendency, band coverage, tuner recommendation
  • Random Wire Analyzer: Avoid lengths, band coverage, match recommendation

📄 Exporting Results

Every tool exports a text file named: 

KG5IEF - <Tool Name>.txt

❗ Common Mistakes

  • Leaving a field blank → enter 0 instead.
  • Not pressing Calculate before exporting.
  • Entering non‑numeric characters into numeric fields.
