User Manual

Complete operating guide for the HF Tools Suite.

1. Introduction

The HF Tools Suite is a browser‑based engineering environment for modeling, analyzing, and exporting antenna system data. It includes tools for verticals, feedlines, DX patterns, NVIS, doublets, and random wires.

2. Critical Rule

All fields must have an entry, even if it is 0.
Blank fields prevent the calculation engine from running.

3. Navigation

Use the left sidebar to select a tool or documentation page. Each tool loads into the main panel with:

  • Input Panel (left)
  • Results Panel (right)
  • Calculate button
  • Download Results button

4. Shared Features

  • Unified HF math engine (wavelength, NVIS, DX, impedance)
  • Consistent export format with callsign header (KG5IEF)
  • SPA router for instant tool switching

5. Tool Overview

  • Vertical Designer: Resonant length, NVIS angle, DX tendency
  • Feedline & Matching: Loss, delivered power, transformed SWR
  • DX Pattern Explorer: Takeoff angle, lobe classification, DX potential
  • NVIS Field Estimator: NVIS angle, coverage radius, suitability
  • Doublet Designer: Impedance tendency, band coverage, tuner recommendation
  • Random Wire Analyzer: Avoid lengths, band coverage, match recommendation

6. Export System

Every tool includes a Download Results button that generates a text file: 

KG5IEF - <Tool Name>.txt

Each file contains your callsign header, timestamp, inputs, and computed results.

7. Troubleshooting

  • Calculations do not run: One or more fields are blank → enter 0 in every field.
  • Export file is empty: Press Calculate before Download Results.
  • Results show “—”: Check for invalid or missing numeric input.
